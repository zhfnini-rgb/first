# AGENTS.md

This repository is used for OpenClaw-related files and experiments.

## Working rules

- Prefer Chinese in user-facing notes unless the task clearly needs English.
- Keep examples minimal and runnable.
- Never commit real secrets, bot tokens, cookies, or personal credentials.
- Prefer environment variables or obvious placeholders for all secrets.
- When editing OpenClaw config, keep the structure simple and explain any non-default setting in `README.md`.
- When adding scripts, include a short usage note.
- Prefer Telegram + Kimi examples unless the user asks for another channel or model provider.

## Safety

- Do not replace placeholders with live credentials.
- Do not add destructive scripts by default.
- Keep automation opt-in and easy to disable.
