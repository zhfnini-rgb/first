# OpenClaw Starter

这是一个用于存放 OpenClaw 相关配置、提示词和工作区文件的起步仓库。

## 仓库里会放什么

- `openclaw.example.json`: OpenClaw 配置示例
- `AGENTS.md`: 给 OpenClaw agent 的仓库级行为说明
- 以后也可以继续补充：技能、自动化脚本、频道配置、部署说明

## OpenClaw 是什么

OpenClaw 是一个多渠道 AI gateway / agent runtime，适合把模型、渠道、自动化和工作区文件放在一起管理。

## 建议的使用方式

1. 把这个仓库 clone 到本地
2. 按你的实际环境修改 `openclaw.example.json`
3. 把适合你的工作规则写进 `AGENTS.md`
4. 再逐步加 Telegram、Kimi、自动化脚本等

## 后续可继续补充

- Tieba / 社媒自动化脚本
- 多 agent 分工配置
