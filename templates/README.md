# Language Templates

这个目录用于放面向不同编程语言项目的 OpenClaw 工作区模板。

目前包含：

- `java/AGENTS.md`
- `python/AGENTS.md`
- `nodejs/AGENTS.md`

## 怎么用

如果你的项目主要是某一种语言：

1. 选择对应语言目录里的 `AGENTS.md`
2. 把内容复制到你项目工作区的 `AGENTS.md`
3. 再按你自己的构建命令、测试命令和目录结构做微调

如果你的项目是多语言混合仓库：

- 可以把通用规则写在根 `AGENTS.md`
- 再把语言专项规则合并进去
- 最好补充你自己项目真实可用的构建和测试命令
